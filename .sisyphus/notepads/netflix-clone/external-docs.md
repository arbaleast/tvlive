
## External Android TV Docs (2026)

- **Android Developers — Use Jetpack Compose on Android TV (Compose for TV)**
  URL: https://developer.android.com/training/tv/playback/compose
  Summary: Introduces Jetpack Compose for TV, explaining how to build TV UIs with Compose and pointing to tv-compose resources and sample knowledge on GitHub.
  Related GitHub: https://github.com/android/tv-codelabs (tv Compose codelabs and samples)

- **Android Developers — Introduction to Compose for TV (Codelab)**
  URL: https://developer.android.com/codelabs/compose-for-tv-introduction
  Summary: A guided codelab showing catalog-browser and details screen implemented with Compose for TV; demonstrates the CatalogBrowser composable pattern.
  Related GitHub: https://github.com/android/tv-codelabs

- **Android Developers — Migrate to Compose for TV (Leanback to Compose)**
  URL: https://developer.android.com/training/tv/playback/leanback/migrate-to-compose
  Summary: Guidance for migrating Android TV apps from Leanback to Jetpack Compose for TV, including hybrid approaches and migration strategies.
  Related GitHub: https://github.com/android/tv-codelabs

- **Android Developers — tv API Release (tv foundation / tv material)**
  URL: https://developer.android.com/jetpack/androidx/releases/tv
  Summary: Official release notes for the TV-related Jetpack libraries (tv-foundation, tv-material), including versions and migration notes.
  Related GitHub: ExoPlayer and Media3 repositories provide complementary playback implementations: https://github.com/google/ExoPlayer

- **Android Developers — Create scrollable layouts for TV (TV lists)**
  URL: https://developer.android.com/training/tv/playback/compose/lists
  Summary: Demonstrates focus-driven scrolling with LazyColumn/Row for TV catalogs and how BringIntoViewSpec can customize scroll behavior.
  Related GitHub: https://github.com/android/tv-codelabs

- **Android Developers — Getting started with ExoPlayer / Media3**
  URL: https://developer.android.com/media/media3/exoplayer/hello-world
  Summary: Step-by-step guide to creating a basic ExoPlayer-based playback flow and attaching it to a UI component; foundations for TV playback via Media3.
  Related GitHub: https://github.com/google/ExoPlayer

- **Android Developers — TV navigation basics**
  URL: https://developer.android.com/training/tv/get-started/navigation
  Summary: Guidelines for D-pad navigation and focus management essential for a great TV UX; tests and consistency across screens.
  Related GitHub: https://github.com/android/tv-codelabs

- **Onboarding to Compose for TV libraries**
  URL: https://android.googlesource.com/platform/frameworks/support/+/HEAD/tv/onboarding.md
  Summary: Official onboarding doc outlining tv-foundation and tv-material usage; points to sample apps and latest library versions.
  Related GitHub: GoogleSource (not GitHub): https://android.googlesource.com/
